KU Website Creator

KU Website Creator

An Easy to Use WYSIWYG Webpage Editor

This was my Software Engineering Final Project. We worked in teams of four and came up with our own project ideas. The only rules were that we had to utilize a software development method (like agile) and use KU's version control system (their utilization of git).

The KU Website Creator allows a user to choose a template of a webpage layout and edit each area to add the content they want. Each area supports text, images, and direct HTML editing (such as linking in YouTube videos or other embedded widgets). When the user was done designing, the file was downloaded to their computer and a webpage with instructions on how to upload the page to their KU given domain space (as well as EECS domain instructions) loaded in their browser.

I was the lead software developer because I had the most experience working in HTML and JavaScript. We also used the systems to test out the software for usability issues with our non-Computer Science friends.

The project can be viewed here.
